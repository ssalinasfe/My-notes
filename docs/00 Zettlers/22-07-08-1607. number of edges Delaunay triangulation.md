# number of edges Delaunay triangulation
**UID**:: 220708160702
**Date**:: 22-07-08
**Creation**::  Friday 08 of July, 2022.  04:55 PM
**Tags**:: 

By [[22-06-11-0206. Euler characteristic]], the number of edges of a [[22-06-06-1806. Triangulation]] is

$|E| = 3n - 3 - h$

The number of triangles is

$|F| = 2n - 2 - h$
